## 剧情简介

"下线了还擅自给我升级什么的，这样的别说是弱保软了都可以说是放置游戏了好伐！"

山田健一绝望了。爽玩的网游停服了。就算去找新游戏来玩，也全都是些弱保软。健一喜欢的，有玩上几万小时价值的作品，几乎都活不下来。

"啥啊……邀请你来到不会结束的游戏，来着？"

健一偶然看到的没有名字的网游。在难易度设定画面毫不犹豫地选择了最高难易度**「地狱模式」**后，就作为**农奴**转生到异世界了！

转生为农奴少年**「亚莲」**的健一，一边完美运用满是谜团的职业**「召唤士」**，一边在既没有攻略也没有论坛的异世界里，探索着迈向最强之路——

## 世界观设定

### 才能系统

这个世界的人们在5岁时会觉醒**「才能」**，才能决定了一个人的人生轨迹。已知的才能包括：

| 才能 | 说明 |
|------|------|
| **召唤士** | 亚莲的才能，只有亚莲能使用的神秘职业 |
| **剑圣** | 克蕾娜的才能，5岁时就能打败成人骑士 |
| **魔道士** | 萨希尔的才能，擅长魔法攻击 |
| **斧士** | 德古拉的才能，近战物理输出 |
| **僧侣** | 基尔的才能，负责治疗与支援 |
| **勇者** | 赫尔米奥斯的才能，传说中的才能 |
| **魔岩将** | 梅露露的才能，能够操纵魔偶 |

### 地狱模式的特点

- **经验值需求100倍**：升级所需经验是普通模式的100倍
- **无攻略无论坛**：没有现成的游戏攻略可参考
- **农奴出身**：初始身份为最底层的农奴
- **废设定世界**：世界规则复杂且不利于生存

## 登场人物

### 主要角色

#### 亚莲（アレン）
- **CV**：田村睦心（动画/广播剧）
- **前世**：山田健一，重度网游玩家
- **才能**：召唤士
- **身份**：小队「颓废玩家」队长

有着「召唤士」才能的少年。前世是重度网游玩家山田健一。在找能让自己玩得下去的游戏时看到的神秘网站里，选择了最高难易度「地狱模式」，在游戏开始的瞬间转生到了异世界。

转生时的身份为农奴，一边走在升级所需经验也要100倍的困难道路上，一边凭着废玩家的毅力和热情成长着。运用着只有亚莲能使用的这份力量，和伙伴及召唤兽一起跨越了众多苦难。

#### 克蕾娜（クレナ）
- **CV**：饭冢麻结（动画/广播剧）
- **才能**：剑圣
- **身份**：小队「颓废玩家」成员

亚莲的青梅竹马。天真烂漫，最喜欢战斗。有着「剑圣」的才能，在5岁时就有了能够打败成人骑士的力量。在入学时与亚莲再会，成为了小队「颓废玩家」的成员。

#### 萨希尔（セシル）
- **CV**：千本木彩花 / 植田佳奈（广播剧）
- **才能**：魔道士
- **身份**：小队「颓废玩家」成员

格兰威尔领地领主的女儿。虽然让亚莲作为侍从跟随左右，但是在那之后和辞去侍从的亚莲一起入学，就变成了对等的关系。

#### 德古拉（ドゴラ）
- **CV**：畠中祐（动画/广播剧）
- **才能**：斧士
- **身份**：小队「颓废玩家」成员

亚莲出生长大的村子里的武器店的儿子。虽然最初对亚莲抱有反感，但是在比剑输给亚莲后就变得亲近了。在入学和与亚莲再会，成为了小队「颓废玩家」的成员。

#### 基尔（キール）
- **才能**：僧侣
- **身份**：小队「颓废玩家」成员

在学园遇见的少年。因为家里的原因，非常执着于金钱。被亚莲邀请，成为了小队「颓废玩家」的成员。

## 作者介绍

### ハム男

日本轻小说作家，以在「成为小说家吧」网站连载的《地狱模式》而闻名。

**创作风格**：
- 以游戏玩家在异世界的冒险为主题
- 融合游戏机制、策略战斗和成长元素
- 注重角色的成长与情感描写
- 善于构建复杂的世界观和策略性战斗场景

## 作品评价

### 亮点分析

1. **独特的设定**：选择「地狱模式」的设定新颖，主角从农奴开始成长，而非一开始就拥有强大力量
2. **策略性战斗**：作为废玩家，主角善于分析和制定策略，战斗场面智斗与武力并重
3. **才能系统**：多样化的才能设定，为角色成长提供了丰富的可能性
4. **团队协作**：小队「颓废玩家」的组建与成长，展现了伙伴羁绊的力量

### 读者评价

## 结语

《地狱模式 ～喜欢挑战特殊成就的玩家在废设定的异世界成为无双～》以其独特的「地狱模式」设定、策略性的战斗描写和丰富的角色群像，在异世界题材中脱颖而出。主角亚莲从农奴到无双的成长历程，以及小队「颓废玩家」成员之间的羁绊，让读者感受到了冒险的热血与友情的温暖。

随着小说的持续连载和动画化企划的推进，我们期待着亚莲能够继续在异世界中书写属于自己的无双传说，揭开这个「废设定」世界的真相。

> 愿每一个面对挑战的人，都能像亚莲一样，凭着废玩家的毅力和热情，在困境中成长，最终实现自己的目标。
